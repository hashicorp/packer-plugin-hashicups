# HashiCups Provisioner

- [Toppings provisioner](/docs/provisioners/hashicups/toppings) - The toppings provisioner is used to add toppings to your order coffee(s).

## How to use this plugin

From Packer v1.7.0, copy and paste this code into your Packer configuration to install this plugin.
Then, run [`packer init`](/docs/commands/init).

```hcl
packer {
  required_plugins {
    hashicups = {
      version = ">= 1.0.1"
      source = "github.com/hashicorp/hashicups"
    }
  }
}
```
