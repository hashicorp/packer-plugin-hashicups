# HashiCups Post Processor

- [Receipt post-processor](/docs/post-processors/hashicups/receipt) - The receipt post-processor is used to
print the receipt of your order.

## How to use this plugin

From Packer v1.7.0, copy and paste this code into your Packer configuration to install this plugin.
Then, run [`packer init`](/docs/commands/init).

```hcl
packer {
  required_plugins {
    hashicups = {
      version = ">= 1.0.0"
      source = "github.com/hashicorp/hashicups"
    }
  }
}
```
